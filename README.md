# Getting Started

## Clone the repository:

```bash
git clone https://github.com/f1nderas/weather-app-next.git
cd weather-app-next
```

## Установка зависимостей:

```bash
npm install
```

## Создайте файл .env на оснвое примера и вставьте токен

```bash
cp .env.example .env
```

## Запускаем:

```bash
npm run dev
```

### Open http://localhost:3000 in your browser.
